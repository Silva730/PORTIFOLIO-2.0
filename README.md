# Portfólio Pessoal — Versão 2

Este é meu portfólio pessoal, desenvolvido como um **site estático** com foco total em frontend.  
O projeto foi criado **por diversão** e para **testar e aprimorar minhas habilidades** em desenvolvimento web.

## Objetivo

Apresentar meus projetos, explorar criatividade visual e experimentar recursos do frontend puro, sem uso de frameworks ou backend.

## Características

- Site estático, leve e rápido
- Interface totalmente construída do zero
- Organização visual voltada para apresentação de projetos
- Uso de elementos multimídia para criatividade e identidade visual

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Conteúdo do portfólio

- Seção de projetos
- Imagens integradas ao layout
- Áudio incorporado para experiência interativa
- Estrutura simples e direta

## Status do projeto

- Versão: **2.0**
- Tipo: Portfólio frontend
- Finalidade: aprendizado, prática e criatividade
